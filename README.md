# The Ultimate Survivor
> 2025 후기 새내기새로배움터 소팀1 게임 프로그램

## 개요
이 프로그램은 2025 후기 새내기새로배움터 소팀1 게임 진행을 위해 제작된 게임 프로그램입니다.

6개의 팀으로 구성되며, 각 팀은 슈퍼 좀비를 피해 생존해야합니다. 생존자들은 생존한 턴 수에 따라 생존 보상으로 점수를 부여받습니다.
슈퍼 좀비에 감염되면 일부 점수를 잃지만, 이후 좀비가 되어 다른 플레이어를 감염시킬 수 있으며, 감염 보상으로 미니게임 수행 결과에 따른 점수를 얻을 수 있습니다.
중간중간 생성되는 각종 특수칸과 이벤트칸을 전략적으로 활용하여 최대한 많은 점수를 확보하는 팀이 승리합니다.

구체적인 메뉴얼은 해당 프로그램의 구글 드라이브 폴더를 참고해주세요.

## 정보
* 실행 환경 : Windows 및 .exe 파일이 구동되는 OS
* 최종 버전 : ver 3.4.2 (release)
* 개발 언어 : C++ 17
* 사용 라이브러리 : SDL2, fpdf

## 설치 방법
우측 Release 페이지에 들어간 후, Download 링크를 클릭하여 설치파일을 다운로드 받을 수 있습니다.   
인증된 프로그램이 아니므로 설치 도중 보안 경고가 뜰 수 있으나, 무시하고 설치하셔도 안전합니다.

## 사용 방법
기본적으로 이동 가능한 타일을 선택하므로써, 말을 이동시킬 수 있습니다.   
숫자를 입력해야하는 경우 숫자키를 단순 입력함으로써 수행할 수 있습니다.

그 외 복잡한 구동은 아래 메뉴얼을 참조해주세요.
* 시작 : 스페이스바를 누르면 타이머가 작동하며 시작
* 말 이동 : 이동하고자 하는 위치의 타일을 클릭하면 해당 위치로 이동
* 현재 이동 턴 : 화면 우측 상단의 점수란에 색으로 표시
* 미니게임 룰렛 : 마우스 좌클릭을 하면 룰렛이 멈춤
* 숫자 입력(미니게임 점수, 팀 번호) : 숫자 입력 후 엔터키 누르기
* 미니게임 슬라이드 : 스페이스바, 방향키 등으로 조작 후 끝나면 엔터로 나감
* 턴 넘기기 / 이벤트 넘기기 : 이동하지 못하는 말의 경우 우클릭 하면 턴을 넘길 수 있음
* 실행취소(undo) / 되돌리기(redo) : z키 / y키
* 전체화면 : f11키
* 화면 그래픽 on/off : t키
* 게임 종료 : esc키
* 게임 초기화 : i키

## 로그
